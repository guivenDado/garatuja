## [HTML] 

HTML é, para mim, uma linguagem de marcação altamente intuitiva. Neste projeto, utilizei diferentes comandos e técnicas com o objetivo de criar um site que homenageia um dos meus personagens favoritos: Sonic! 🦔💙
